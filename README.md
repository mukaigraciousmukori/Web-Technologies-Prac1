# Web-Technologies-Prac1

A comprehensive, responsive portfolio website showcasing modern web design and development practices.

## Project Description

Web-Technologies-Prac1 is a fully responsive personal portfolio website built to demonstrate proficiency in web technologies and front-end development. The site features:

- **Home Page**: A welcoming hero section with a professional introduction and call-to-action button
- **About Page**: Detailed biography with a visual timeline of work experience and an interactive skills showcase with progress bars
- **Projects Page**: A grid-based portfolio displaying completed projects with technology tags and links to live demos and source code
- **Contact Page**: A fully functional contact form with social media links for easy communication

The website is optimized for all device sizes, from mobile phones to large desktop displays, ensuring a seamless user experience across all platforms.

## Technologies Used

### Frontend
- **HTML5**: Semantic markup with proper structure and accessibility
- **CSS3**: Modern styling with:
  - Flexbox for responsive layouts
  - CSS Grid for project cards
  - Media queries for mobile-first responsive design
  - CSS animations (@keyframes) for skill bar filling effects
  - `clamp()` for fluid typography and spacing
- **JavaScript**: Basic interactivity and form handling
- **Font Awesome**: Icon library for visual enhancement
- **Google Fonts**: Poppins font family for modern typography

### Design & UX
- Dark theme color scheme for reduced eye strain
- Responsive breakpoints at 480px (mobile), 768px (tablet), and 1400px+ (desktop)
- Touch-device optimizations for better mobile experience
- Smooth transitions and hover effects for interactive feedback

## Setup Instructions

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server-side setup required

### Steps to Run Locally

1. **Clone or Download the Repository**
   ```bash
   git clone https://github.com/mukaigraciousmukori/Web-Technologies-Prac1.git
   cd Web-Technologies-Prac1
   ```

2. **Open the Website**
   - Simply open `index.html` in your web browser
   - Or use a local server (recommended for better performance):
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Python 2
     python -m SimpleHTTPServer 8000
     
     # Using Node.js (if installed)
     npx http-server
     ```

3. **Access the Website**
   - Local file: Open `index.html` directly in your browser
   - Local server: Navigate to `http://localhost:8000` in your browser
